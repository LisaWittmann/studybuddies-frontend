# frontend

## Project setup
```
npm install
```

### Compiles and hot-reloads for production
```
npm run serve
```
### Compiles and hot-reloads for development
```
npm run serve-dev
```

### Compiles and minifies for production
```
npm run build
```

### Compiles and minifies in development mode
```
npm run build-dev
```

### Run Unit tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```
